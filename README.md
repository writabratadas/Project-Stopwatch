# ⏱️ Stopwatch

A simple and responsive stopwatch application to track elapsed time.

## 🚀 Features

- Start the stopwatch
- Stop the stopwatch
- Reset the stopwatch
- Display elapsed time in minutes, seconds, and milliseconds
